 Week2 3D 坐标变换（MVP矩阵）实验报告
一、实验目标
1. 深入理解 3D 空间中 MVP（模型-视图-投影）坐标变换 完整流程
2. 独立实现模型旋转、视图平移、透视投影三大核心矩阵
3. 掌握 Taichi 框架的矩阵运算与 GPU 并行编程基础
4. 将 3D 空间三角形通过矩阵变换渲染为 2D 屏幕图形

 二、实验理解与设计思路
本次实验的核心是 3D 坐标到 2D 屏幕的映射：
1. 模型变换：让 3D 三角形绕 Z 轴旋转，实现动态交互效果
2. 视图变换：将相机移动到坐标原点，统一观察视角
3. 投影变换：将 3D 透视锥体压缩为 2D 平面，模拟人眼视觉
4. 透视除法 + 视口映射：将标准坐标转换为屏幕可显示的坐标

全程遵循 列向量右乘规则 ，严格按照实验要求实现矩阵运算。

三、核心代码关键环节解析
 1. 模型变换矩阵 `get_model_matrix`
- 功能：接收旋转角度，生成 绕 Z 轴旋转 的 4×4 矩阵
- 关键：角度转弧度，使用三角函数构建标准旋转矩阵
- 作用：控制三角形在 3D 空间中的旋转姿态

 2. 视图变换矩阵 `get_view_matrix`
- 功能：将相机平移至世界坐标原点
- 关键：通过平移矩阵抵消相机位置，实现视角转换
- 作用：统一观察坐标系，符合图形学标准渲染流程

 3. 透视投影矩阵 `get_projection_matrix`
- 功能：两步实现投影（透视挤压 + 正交投影）
- 关键：根据视场角计算视锥体边界，处理 Z 轴符号
- 作用：将 3D 空间投影到 2D 平面，实现近大远小效果

4. 核心渲染流程
   1. 顶点应用 MVP 矩阵变换
   2. 透视除法（齐次坐标除以 w 分量）
   3. 映射到屏幕 `[0,1]` 坐标空间
   4. Taichi GUI 绘制彩色线框三角形

四、实验效果演示
1. 程序启动后弹出 **700×700** GUI 窗口
2. 显示**红、绿、蓝三色线框三角形**
3. 按 **A 键**：三角形逆时针旋转
4. 按 **D 键**：三角形顺时针旋转
5. 按 **ESC 键**：关闭窗口退出程序
